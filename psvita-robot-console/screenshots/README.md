# Screenshots

This folder contains screenshots for the PS Vita Robot Console application.

## Required Screenshots

Please add the following screenshots to help users understand the application:

1. **main_interface.png** - Main control screen showing:
   - Green tactical HUD with crosshair reticle
   - Live telemetry data (battery, ping, WiFi signal)
   - Camera feed display
   - LIN/ANG sliders
   - Robot status indicators

2. **config_screen.png** - Configuration interface showing:
   - Tabbed robot configuration
   - IP address input fields
   - Camera port and endpoint settings
   - PS Vita IME keyboard

3. **emergency_stop.png** - Emergency stop activated showing:
   - Red overlay effect
   - Warning indicators
   - Haptic feedback visual cues

4. **robot_selection.png** - Multi-robot management showing:
   - Robot selection interface
   - Multiple robot tabs
   - Connection status for each robot

## How to Capture Screenshots

1. Run the PS Vita Robot Console on your PS Vita or Vita3K
2. Use the Cross button to save screenshots to `ux0:data/RobotConsole/screenshots/`
3. Copy the screenshots to this folder and rename them according to the list above
4. Ensure screenshots are clear and show the key features mentioned

## Image Guidelines

- Use PNG format for best quality
- Recommended resolution: 960x544 (PS Vita native resolution)
- Ensure UI elements are clearly visible
- Avoid personal information in screenshots
